# Retail Sales Intelligence: SQL & Python Analysis

## Project Overview
This project involves a deep-dive analysis of retail transaction data to identify key revenue drivers and customer behavioral patterns. By cleaning and processing over 2,000 records, I transformed raw data into actionable business intelligence.

## Key Insights Found
* **Top Revenue Category:** Electronics led all categories with **$313,810** in total sales.
* **Customer Segmentation:** Identified a "VIP" segment (Age 18-35) that contributes to **16.3%** of total revenue.
* **Operational Efficiency:** Discovered that **52.2%** of transactions occur after 5:00 PM, suggesting a need for shifted staffing hours.

## Technical Toolkit
* **Language:** Python (Pandas, NumPy)
* **Environment:** Google Colab / Jupyter Notebook
* **Database:** SQL for data aggregation and window functions
* **Concepts:** Data Cleaning, Time-Series Analysis, and Descriptive Statistics

## How to Use
1. Open `retail_db.ipynb` to view the step-by-step analysis and code.
